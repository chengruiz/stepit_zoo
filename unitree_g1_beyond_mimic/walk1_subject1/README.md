# unitree_g1_beyond_mimic_walk1_subject1

Model trained in the `Tracking-Flat-G1-Wo-State-Estimation-v0` environment of [whole_body_tracking](https://github.com/HybridRobotics/whole_body_tracking), to track the `walk1_subject1` motion of [LAFAN1 Dataset](https://huggingface.co/datasets/lvhaidong/LAFAN1_Retargeting_Dataset).
